Why draw

Draw, don't describe

No typing, no prompt-craft. Circle exactly what you mean and let the position do the talking — the fastest way to ask about anything onscreen.

Implicit questions

It infers what you're asking from what you circled and where — no need to spell out the question.

Streaming answers

Text writes onto the sticky note as it's generated — you start reading before it finishes.

Swappable AI backends

Bring your own API key and point it at any provider — one config line switches the whole engine. Groq for speed, Ollama for local, whatever fits.

Cross-platform

Native builds for macOS, Linux, and Windows — same hotkey, same flow, everywhere you work.

Save as markdown

Every answer exports to clean markdown — searchable, shareable, yours to keep.

Bring your own brain

Swap the model backend with one line. Your key, your provider.
